Simplify Your Invoice Uploads

The BIF Uploader application streamlines the process of uploading Billing Information Format (BIF) files to Invoice Cloud with automatic encoding and validation.

Download Now Learn More

Key Features

Automatic BASE64 Encoding

Intelligently detects file formats and automatically handles BASE64 encoding - no manual encoding required!

Format Validation

Built-in validation ensures your BIF files meet proper format requirements before upload.

Credential Management

Securely store multiple credential profiles for different Invoice Cloud environments.

Scheduled Uploads

Set up one-time or recurring scheduled tasks to automatically upload your invoice files when needed.

BIF File Editing

Edit your BIF files directly within the application - make quick changes without needing external tools.

Multiple Upload Methods

Choose between SOAP 1.1, SOAP 1.2, or HTTP POST upload methods to suit your needs.

Save Encoded Files

Export and save encoded BIF files for future use or troubleshooting.

Detailed Logging

Track all operations with comprehensive logging and detailed error messages.

How It Works

1

Select Your BIF File

Browse and select your BIF format file. The application automatically detects if your file is already BASE64 encoded and handles encoding for you if needed - no manual encoding required!

2

Edit If Needed

Use the built-in editor to make any necessary changes to your BIF file directly in the application. No need for external text editors!

3

Enter Credentials

Provide your Invoice Cloud credentials (Biller GUID and Web Service Key) or select a saved profile.

4

Choose Upload Method

Select your preferred upload protocol (SOAP 1.1, SOAP 1.2, or HTTP POST) and decide whether to upload immediately or schedule for later.

5

Upload and Monitor

Click upload to process immediately or set up a one-time/recurring schedule. Track all progress through the detailed operation log.

Frequently Asked Questions

What is BASE64 encoding and why is it needed?

BASE64 encoding converts binary data into a text format that can be safely transmitted over systems that handle text only. While Invoice Cloud offers this as an optional endpoint to enhance secure file transmission, BIF Uploader automatically handles this encoding for you, so you don't need to worry about manually encoding your files.

Can I schedule my invoice uploads?

Yes! BIF Uploader offers flexible scheduling options. You can set up one-time scheduled tasks for future uploads or recurring tasks (daily, weekly, monthly) to automate your regular invoice processes. The application will handle the uploads automatically according to your schedule, even when you're not at your computer.

Can I edit my BIF files in the application?

Yes! BIF Uploader now includes a built-in file editor that allows you to make changes to your BIF files directly within the application. Simply load your file and use the editor to make any necessary modifications before uploading.

Do I need to encode my files before uploading?

No! BIF Uploader intelligently detects whether your file is already encoded. If it's not, the application will automatically encode it for you during the upload process. You can simply select your raw BIF file and the application handles everything else.

How do scheduled uploads work?

To schedule an upload, simply prepare your BIF file as usual, then instead of clicking "Upload Now," select "Schedule Upload" and choose your preferred timing. You can set a one-time future date/time or configure a recurring schedule (daily, weekly, monthly). The application will run in the background and execute the upload task at the scheduled time. All activities will be logged for your review.

Can I save the encoded file for future use?

Yes, you can click the 'Save Encoded File' button to save the BASE64 encoded version of your BIF file. This can be useful for debugging or for uploading through other means.

How do I know if my upload was successful?

A successful upload will display a green results panel with a success message, file name, and File GUID. This information will also be logged in the Operation Log section of the application.

Can I use the same BIF file format for all billing types?

Yes, the BIF format is standardized across Invoice Cloud, but different billing types may require specific fields to be included. Consult the Invoice Cloud documentation for your specific industry requirements.

Are my credentials stored securely?

Yes, credentials are stored locally on your computer in an encrypted format. You can also choose not to save credentials at all if your security policies require it.
